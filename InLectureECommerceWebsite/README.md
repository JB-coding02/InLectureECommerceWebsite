# Justin's Game Store
This is a website to learn the fundamentals<br>
of ASP.NET Core MVC (Model View Controller).

## Upcoming Features
- Styling with Bootstrap 5
- CRUD (Create/Retrieve/Update/Delete) functionality with Entity Framework Core
- Login/Register
- Shopping Cart

## How to get started
- Clone or download the repository from GitHub
- .NET 10 SDK
- Recommend to use Visual Studio 2026
- Open in VS 2026and press the run button

## Technologies/Frameworks used
- ASP.NET Core MVC with C#
- Entity Framework Core
- Bootstrap 5
- HTML/CSS/JS
